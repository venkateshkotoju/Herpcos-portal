# HerPCOS — Claude Code Project Brief

## Business Overview

**Project:** HerPCOS
**Type:** AI-powered Niche Expert Portal
**Launch niche:** PCOS (Polycystic Ovary Syndrome)
**Long-term vision:** Women's Hormonal Health Platform

### Future categories (in order)
1. PCOS
2. Fertility
3. Hormonal Weight Loss
4. Pregnancy
5. Postpartum
6. Menopause

---

## Technical Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **AI:** OpenAI API
- **Hosting:** Vercel
- **Version control:** GitHub
- **Domain:** HerPCOS.com

### Future flexibility (not yet in use)
- Supabase / Appwrite / Convex for auth and database

---

## Current Status

- Live at HerPCOS.com with HTTPS
- AI chatbot working with OpenAI integration
- Deployed on Vercel
- Current phase: **Traffic and validation**

## Product Vision

### MVP (current)
- AI chatbot

### Future (only when justified by real users)

---

## Product Principles

Always:
- Prioritize trust above all else
- Use evidence-based information only
- Never diagnose — always include medical disclaimers
- Keep explanations beginner-friendly and accessible

---

## Development Philosophy

- **Build MVP first. Do not overengineer.**
- Prefer the simplest solution that works
- Plain text and simple components over complex abstractions
- Only add complexity when justified by real user needs
- Recommend changes only when there is a clear benefit

### Priorities (in order)
1. User value
2. Fast iteration
3. SEO
4. Trust

---

## Growth Strategy

### Current priorities (in order)
1. Improve chatbot quality
2. Create SEO content
3. Get first users
4. Validate engagement

Do not prioritize monetization until users and traffic exist.
